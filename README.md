# Блог на Vue-3
Блог, в котором с API получаются данные о постах. Для их отображения используются разные технологии, разные способы реальзации пагинации.

## Технологии:
* Vue-3
* HTML
* JS
* CSS
* Vue-route
* Vue-composition
* Vuex
* Vue store
